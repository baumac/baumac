My Go solutions for [Advent of Code 2025](https://adventofcode.com/2025).
## Usage

To find the solution for a given day:
1. open a terminal and cd into the desired day's directory i.e `cd ./2025/day-01`
2. copy your input into the desired day's input.txt file i.e `pbpaste > input.txt`
3. run `go run dayXX.go` i.e `go run day01.go`

### Running tests

To test the solution for a given day:
1. open a terminal and cd into the desired day's directory i.e `cd ./2025/day-01`
2. run test with `go test`

## Layout

In each day you will find:
- README.md: the problem
- input.txt: the provided input
- dayXX_test.go: a test for the solution
- dayXX.go: the solution
